Get Started

2-Minute Quick Setup

Add the library directly to your static HTML or web application using simple, clean tags.

Integration Steps

1

Include Stylesheet & Script

Add the single css bundle to your page's <head> and the single js bundle right before closing </body>.

2

Create Target Input Box

Add a standard text input element to your HTML form with a selector target (e.g. id="nepali-date").

3

Initialize via JavaScript

Instantiate a new class instance with custom configuration options such as theme, language, and selection mode.

HTML / JavaScript 
<!-- 1. Include library assets -->
<link rel="stylesheet" href="dist/nepali-datepicker.css">

<!-- 2. Add input element -->
<input type="text" id="nepali-date" placeholder="Select Nepali Date">

<!-- 3. Include JS and initialize -->
<script src="dist/nepali-datepicker.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    new NepaliDatePicker('#nepali-date', {
      theme: 'glassmorphism',
      lang: 'ne', // 'ne' (Nepali) or 'en' (English)
      showAdDate: true,
      onChange: function(date) {
        console.log('Selected date details:', date);
      }
    });
  });
</script>

API Configuration Presets

Strict API Configurations

Interactive demonstration of official parameters, disablers, selection modes, and utility helper functions.

Dashed Year-Month-Day

dateFormat: 'YYYY-MM-DD'

Standard ISO-like date mapping where components are separated by dashes. Perfect for standard database synchronization fields. Outputs values like 2083-03-11.

Slashed Year-Month-Day

dateFormat: 'YYYY/MM/DD'

Traditional directory-style presentation separating the year, month, and day components with slashes. Outputs values like 2083/03/10.

Dotted Year-Month-Day

dateFormat: 'YYYY.MM.DD'

Sleek dot-separated layout frequently favored in modern web applications. Outputs values like 2083.03.02.

Dashed Day-Month-Year

dateFormat: 'DD-MM-YYYY'

European/Commonwealth dashed representation format starting with the day value first. Outputs values like 12-03-2083.

Slashed Day-Month-Year

dateFormat: 'DD/MM/YYYY'

Highly recognizable format style across Commonwealth contexts, utilizing slash characters. Outputs values like 15/03/2083.

Dotted Day-Month-Year

dateFormat: 'DD.MM.YYYY'

Elegant European-style formatting utilizing dotted separators starting with the day first. Outputs values like 15.03.2083.

Dashed Month-Day-Year

dateFormat: 'MM-DD-YYYY'

Dashed presentation style starting with the month value, followed by days and year. Outputs values like 03-15-2083.

Slashed Month-Day-Year

dateFormat: 'MM/DD/YYYY'

Standardized formatting layout matching American date preferences, using slashes. Outputs values like 03/15/2083.

Gregorian AD Long format

dateFormat: 'DD MMMM YYYY'

Converts selected BS date to Gregorian calendar (AD) and outputs long-form values like 01 February 2035.

Minimum Date Limit

minDate: { year: 2083, month: 3, day: 3 }

Prevents scheduling in the past. All calendar cells prior to the configured min date are blocked out and disabled.

Maximum Date Limit

maxDate: { year: 2083, month: 3, day: 13 }

Prevents excessive scheduling into the future. All calendar cells after the configured maximum limit are locked.

Bounded Range Window

minDate & maxDate

Sets strict boundaries on both sides, creating a valid selection window of exactly 11 days (March 3 to March 13, 2083).

Selection Callback Event

onSelect: function(date)

Triggers custom application workflows when a date is selected. This demo logs parameters to the console and alerts the user.

Disable Today's Date

disableToday: true

Blocks users from scheduling actions on the current day. The today cell remains greyed out and unselectable.

Disable Specific Date

disableDates: ['2083-3-9']

Block out specific dates like holidays or fully-booked slots. Specifically blocks selection of the date 2083-03-09.

Disable Multiple Dates

disableDates: [...]

Passes an array of custom dates to disable multiple selections at once (disables the 1st, 15th, and 28th of the month).

Disable Relative Past

disableDaysBefore: 5

Prevents selecting dates older than a rolling relative buffer. Disables all days falling more than 5 days before today.

Disable Relative Future

disableDaysAfter: 5

Restricts booking too far ahead. Disables selection of all dates falling more than 5 days in the future.

Relative Selection Range

disableDaysBefore & After

Combines rolling limits relative to today, locking the selectable range to a rolling ±5 day window.

Slide Transition

animation: 'slide'

Configures the dropdown animation to slide up from the bottom with a smooth spring bounce.

Fade Transition

animation: 'fade'

Smooth fade-in opacity animation overlay for modern, minimalistic design tastes.

Range Select Mode

range: true

Select consecutive calendar date boundaries directly in a single picker element with a hover preview.

Multiple Dates Select

multiple: true

Click to add or remove multiple non-adjacent dates. Selected dates are displayed as a comma-separated list.

Unicode Devanagari

unicodeDate: true

Translates all digits in the date picker interface and form output to Nepali Unicode (e.g. २०८३-०३-१५).

English Interface

language: 'english'

Switches the months, weekdays, and headers of the Bikram Sambat picker to English translation text.

Accuracy Redefined

BS ↔ AD Calendar Date Converter

Our open-source conversion engine is powered by validated static calendar metadata spanning the years 1970 BS to 2100 BS (1913 AD to 2043 AD). Includes public holidays and festival indicators.

  • Bidirectional Conversion

    Instantly map Bikram Sambat (BS) to Gregorian Calendar (AD) and vice-versa with 100% accuracy.

  • Standalone Widget Component

    Drop the widget anywhere on a dashboard or landing page as an independent conversion widget.

  • Dynamic Meta Insights

    Returns day of the week, total days in the month, leap year status, and days difference calculation.
